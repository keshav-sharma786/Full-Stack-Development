# ID Attribute vs Class Attribute

1. ID

1) It is used to target elements uniquely.
2) Multiple elements cannot have same id.
3) One element can have only 1 id's value.
4) To target an element using id in CSS we use # (hash)

2. CLASS

1) It is used to target multiple elements at a time.
2) Multiple elements can have same class.
3) One element can have multiple class's value
4) To Target an element using class in css we use . Dot
